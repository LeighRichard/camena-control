## 项目概述

### 系统简介

这是一个基于 **Jetson Nano + STM32 + 奥比中光深度相机** 的智能相机控制系统，实现了：
- 🎯 自动目标检测与跟踪
- 👤 人脸识别与自动拍摄
- 🤖 视觉伺服控制
- 📱 多平台远程控制（桌面端 + 移动端）
- 🌐 4G/WiFi 远程访问
- 📊 实时监控与告警

### 核心特性

- **三轴运动控制**: Pan（水平）、Tilt（俯仰）、Rail（滑轨）
- **深度感知**: 基于结构光深度相机的 3D 定位
- **智能跟踪**: YOLOv5/v8 目标检测 + 卡尔曼滤波
- **自动拍摄**: 路径规划 + 质量评估 + 自动触发
- **远程控制**: Web/桌面/移动端多平台支持
- **安全保护**: 限位检测 + 看门狗 + 紧急停止

### 硬件连接说明

#### 1. Jetson Nano 连接
- **USB 3.0** → 奥比中光深度相机
- **UART (GPIO14/15)** → STM32 (USART1)
- **以太网/WiFi** → 网络连接
- **USB 4G 模块** → 移动网络（可选）
- **电源** → 5V/4A DC 适配器

#### 2. STM32F407VET6 连接
- **USART1 (PA9/PA10)** → Jetson Nano UART
- **TIM1/2/3** → 步进电机 PWM 输出
- **GPIO** → 限位开关输入
- **GPIO** → 紧急停止按钮
- **电源** → 5V (从 Jetson 或独立供电)

#### 3. 步进电机驱动
- **步进电机驱动器** (如 TB6600/DM542)
- **PUL+/PUL-** ← STM32 PWM 输出
- **DIR+/DIR-** ← STM32 方向控制
- **ENA+/ENA-** ← STM32 使能控制
- **电源** → 24V/36V DC (根据电机规格)

### B. 通信协议详细说明

#### 帧格式
```
[HEAD] [SEQ] [LEN] [CMD] [DATA] [CRC16] [TAIL]
  2B    2B    2B    1B   0-255B   2B      2B
```

#### 命令列表
| 命令码 | 名称 | 数据格式 | 说明 |
|--------|------|---------|------|
| 0x01 | MOVE | axis(1B) + steps(4B) + speed(2B) | 移动指定轴 |
| 0x02 | STOP | axis(1B) | 停止指定轴 |
| 0x03 | STOP_ALL | - | 停止所有轴 |
| 0x04 | HOME | axis(1B) | 归零指定轴 |
| 0x05 | GET_POS | axis(1B) | 查询位置 |
| 0x06 | SET_SPEED | axis(1B) + speed(2B) | 设置速度 |
| 0x07 | GET_STATUS | - | 查询状态 |
| 0x08 | SET_PARAM | param_id(1B) + value(4B) | 设置参数 |
| 0x09 | GET_PARAM | param_id(1B) | 查询参数 |
| 0x0A | EMERGENCY_STOP | - | 紧急停止 |
