## 执行摘要

本系统是一个基于 NVIDIA Jetson Nano 和 STM32 的嵌入式相机位置控制与自动拍摄系统。经过全面的开发和测试，所有核心功能已成功实现并通过验证。

### 系统组成

1. **Jetson Nano 端** (Python) - 主控制器
2. **STM32F407 端** (C) - 运动控制器
3. **Web 应用** (Flask + JavaScript) - 浏览器访问
4. **手机应用** (Flutter) - iOS/Android 客户端
5. **桌面应用** (Electron + React) - Windows/macOS/Linux 客户端
