## 📊 项目概览

这是一个基于 NVIDIA Jetson Nano 和 STM32 的专业级嵌入式相机位置控制与自动拍摄系统，支持深度学习目标检测、自动拍摄流程和多端远程控制。

### 核心特性

- 🎯 **智能目标检测** - YOLOv5/YOLOv8 + TensorRT 加速
- 🤖 **精确运动控制** - PID + S曲线速度规划
- 📸 **自动拍摄流程** - 路径规划 + 时序控制
- 🌐 **多端访问** - Web + 手机 + 桌面应用
- 📡 **远程控制** - WiFi 热点 + 4G/5G 网络
- 🔒 **安全保护** - 限位检测 + 急停 + 看门狗

## 🚀 部署指南

### 快速开始

#### 1. Jetson Nano 端
```bash
cd jetson
pip install -r requirements.txt
python -m pytest tests/  # 运行测试
python src/web/app.py    # 启动服务
```

#### 2. STM32 端
```bash
cd stm32
mkdir build && cd build
cmake ..
make
# 烧录固件到 STM32
```

#### 3. Web 访问
```
打开浏览器访问: http://192.168.4.1:5000
```

#### 4. 手机 APP
```bash
cd mobile_app
flutter pub get
flutter run
```

#### 5. 桌面 APP
```bash
cd desktop_app
npm install
npm run dev
```
